# 反思总结
## 毫无疑问抽象是会降低程序性能的。我们自己封装的Vector类性能上比std::vector要优秀很多，但是略微劣于原生的数组。在使用了相同优化算法的情况下肯定是原生的数组更快。
## cpp中的有“抽象零惩罚”这么一说。但是需要开动脑筋来进行设计。这也是导致我们后期乏力的一个原因。
